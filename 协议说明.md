# 协议说明

### 传输格式

- 本游戏协议采用纯字符串json格式传输
- 如：
      {
      "cmd" : "THE CMD",
      "ver" : "VERSION",
      "param" : {
        "param1" : "value1",
        "param2" : "value2",
        }
      }
- 其中：
  - cmd代表指令名称，并映射到对应的处理方法
  - ver代表协议版本号，当前协议版本号为1.0
  - param代表协议参数，内容也是json结构


### 协议详解

- Login，登录
